Holter monitor 2 channel untuk monitoring ECG jangka panjang dengan ESP32 DevKit V1. Menggunakan 2x AD620 untuk dual channel acquisition dan TL084 untuk filtering. Dilengkapi dengan shield khusus ESP32 untuk akuisisi data dan wireless transmission.

ECG Circuit

Rangkaian elektrokardiogram (ECG/EKG) untuk merekam aktivitas listrik jantung. Menggunakan instrumentation amplifier AD620 untuk akuisisi sinyal, TL084 untuk filtering, dan OP07 sebagai DRL (Driven Right Leg) circuit.

Spesifikasi:
‚Ä¢ IC Utama: AD620 (Instrumentation Amp)
‚Ä¢ Filter: TL084 (Quad Op-Amp)
‚Ä¢ DRL Circuit: OP07 (Precision Op-Amp)
‚Ä¢ Gain: Adjustable (via resistor)
‚Ä¢ Bandwidth: 0.5Hz - 150Hz (typical ECG)
‚Ä¢ Aplikasi: Medical monitoring, biofeedback

IC 7660 - Voltage Inverter

Rangkaian pembuat tegangan negatif menggunakan IC 7660. IC ini dapat mengkonversi tegangan positif menjadi tegangan negatif dengan arus output hingga 100mA. Sangat berguna untuk power supply op-amp.

Spesifikasi:
‚Ä¢ Input: +5V hingga +12V DC
‚Ä¢ Output: -5V hingga -12V DC
‚Ä¢ Arus Max: 100mA
‚Ä¢ Komponen: IC 7660, kapasitor
‚Ä¢ Aplikasi: Dual supply untuk op-amp

üì• Cara Menggunakan

  1. Klik tombol "Download Schematic" atau "Download Board" di atas
  2. Buka Autodesk Eagle di komputer Anda
  3. Buka file .sch untuk melihat skematik atau .brd untuk melihat layout PCB
  4. Anda dapat mengedit dan menggunakan desain sesuai kebutuhan
  5. Untuk mencetak PCB, export file .brd ke format Gerber melalui menu CAM Processor

Atau clone seluruh repository:

git clone https://github.com/aaabdussalamm-boop/eagle-catalog.git

üí° Tips Integrasi:

  • Gunakan IC 7660 sebagai power supply negatif untuk rangkaian PCG Quad atau ECG
  • Rangkaian ECG dan Holter memerlukan isolated power supply untuk keamanan pasien
  • Semua rangkaian op-amp memerlukan dual supply (¬±V) untuk operasi optimal
  • Holter AD dapat mengirim data ECG via WiFi/Bluetooth ESP32 untuk monitoring real-time
  • Untuk aplikasi portable, gunakan battery dengan voltage regulator yang stabil
