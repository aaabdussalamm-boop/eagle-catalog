IC 7660 - Voltage Inverter

Rangkaian pembuat tegangan negatif menggunakan IC 7660. IC ini dapat mengkonversi tegangan positif menjadi tegangan negatif dengan arus output hingga 100mA. Sangat berguna untuk power supply op-amp.

Spesifikasi:

‚Ä¢ Output: -5V hingga -12V DC
‚Ä¢ Arus Max: 100mA
‚Ä¢ Komponen: IC 7660, kapasitor
‚Ä¢ Aplikasi: Dual supply untuk op-amp

üì• Cara Menggunakan

  1. Klik tombol "Download Schematic" atau "Download Board" di atas
  2. Buka Autodesk Eagle di komputer Anda
  3. Buka file .sch untuk melihat skematik atau .brd untuk melihat layout PCB
  4. Anda dapat mengedit dan menggunakan desain sesuai kebutuhan
  5. Untuk mencetak PCB, export file .brd ke format Gerber melalui menu CAM Processor
